# vantasy-G3
bot-whatsapp G3
# Langkah² install bot dan Module bot
```bash
$ pkg update && pkg upgrade -y
$ pkg install nodejs python git -y
$ git clone https://github.com/CandraMc1909/vantasy-G3.git
$ cd Vantasy
$ npm init -y
$ npm install @whiskeysockets/baileys pino qrcode-terminal
```
# Raning Bot
```bash
$ node index
```
# Add token
```bash
$ cd Vantasy/database
$ python craft.py
$ input panjang token
$ input banyak token
$ y (save ke harian.txt, token.txt, atau vocer.txt)
```
